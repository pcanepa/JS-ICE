How can I show to people my simulated vibrational frequencies throughout a friendly HTML page?

This page leads the user in the creation (step by step) of web pages showing outputs of frequency calculations.

  1. Load in J-ICE the output files produced with CRYSTAL09 (06, 03), VASP or QuantumEspresso (package PHon).
    Watch the video here.
  2. Modify the structure as you wish with the options available in J-ICE i.e.: change atom colour, change bond connectivity, change background, make polyedra, etc..

    3. Watch the video here.

    You should be at this stage now

  3. Create a new folder through the file manager of your operative system as shown in the picture below. This is meant to contain the web-page and the other necessary files. The structure of your folder must follow the follow scheme reported below.

    *Files: current.spt, output.dat and webpage.html will be created during the automatic procedure.
    • current.spt contains info about the previous changes done on the actual structure with J-ICE.
    • output.dat is the simulation output.
    • webpage.html is obviously the web page.
  4. Nested into the web-page folder you must also make a new folder called jmol (all small letters, see Picture above).
  5. Download and uncompressJmol into the jmol folder (as created in step 4) form here.

    6. WARNING: Make sure to rename the uncompressed folder in jmol

  6. Click on the button below save HTML frequencies.

    7. WARNING: Avoid filename changes during the export operation
